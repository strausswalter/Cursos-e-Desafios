Projeto Infnetgrid

O "Infnet Grid" foi idealizado com a intenção de que o aluno veja o desenvolvimento de um projeto desde o JavaScript, passando pelo PHP e finalizando no AJAX.

Desta forma, todos os conceitos aprendidos em JavaScript serão praticados no curso de PHP e AJAX. Todos os conceitos aprendidos em PHP serão praticados em AJAX.

Nem tudo funciona. Vocês verão isso no decorrer dos três cursos.

Significa dizer que não se aprendo o "todo" num primeiro momento, mas que esse "todo" está integrado entre os três cursos... Daí o nome "Grid".

Esperamos que gostem do que está sendo projetado. Bom Projeto para todos.
